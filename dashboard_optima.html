Contexto de esta instalación
Las luminarias suministradas fueron instaladas a una altura superior a la recomendada para el sensor estándar incluido. Esto ha generado que el sensor no detecte el paso de personas en algunos momentos, activándose la iluminación de forma intermitente o dejando de hacerlo cuando se requiere.

Esto no es un fallo del equipo: el sensor opera dentro de su especificación técnica para alturas de hasta 15 m. El factor determinante fue la altura de montaje final elegida durante la instalación.
💡
Solución disponible
Optima Energía cuenta con sensores específicos para alturas de hasta 20 m. Para la próxima fase o para ajustar esta instalación, se puede reemplazar únicamente el módulo sensor, sin necesidad de cambiar las luminarias. Esto garantiza una detección confiable y maximiza el ahorro energético por encendido automático.
